# DIAS - Digital Indsamling Af Studenteropgaver

for at installere DIAS skal du have følgende installeret

* Ruby 1.9.3
* Rails 3.1.x
* hydra-jetty
* 
* 


For starte DIAS
* git clone dias
* git clone hydra-jetty
* rake db:migrate
* rake hydra:jetty:config 
* rake jetty:start
* rails server
* gå til localhost:3000
